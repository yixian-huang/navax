# nav.ax 站点网格 Logo 导出

- favicon.svg — 矢量源文件（墨底方版，推荐 <link rel="icon" type="image/svg+xml">）
- favicon-16/32/48.png — 浏览器标签栏
- favicon-64/128.png — 书签/历史列表
- favicon-180.png — apple-touch-icon
- favicon-192/512.png — PWA manifest
- mark-light.svg — 透明底 mark，浅色界面内使用（墨色 + 苔绿 #4A6B52）
- mark-dark.svg — 透明底 mark，深色界面内使用（纸白 + 苔绿 #6F9C7D）

HTML 引用：
```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
<link rel="apple-touch-icon" href="/favicon-180.png">
```
